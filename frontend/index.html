How PCs are registered

PCs are automatically registered when the Bazooka Agent is run on a PC. You cannot manually create PCs from this dashboard.

Deploy Agent

Download the agent files and run them on any PC you want to monitor:

  • Copy the entire agent folder to the target PC
  • Run run-agent.bat (Windows) or run-agent.command (macOS/Linux)
  • The PC will automatically appear in your dashboard
